# 🚚 Factory-to-Customer Shipping Route Efficiency Analysis for Nassau Candy Distributor

## 📌 Overview

This project is a **data-driven logistics and supply chain analytics dashboard** built using Streamlit. It focuses on analyzing and optimizing shipping routes from factories to customers, helping identify inefficiencies, bottlenecks, and opportunities for improvement.

The system provides actionable insights into **lead time distribution, regional performance, and route optimization**, enabling smarter decision-making in supply chain operations.

---

## 🎯 Objectives

* Analyze shipping performance across regions and routes
* Identify bottlenecks and delays in logistics
* Optimize factory-to-customer allocation
* Improve delivery efficiency and reduce lead time

---

## 🚀 Features

*  ⚙️     Factory Optimization Simulator  
* 📉  What-If Scenario Analysis  
* 🔁  Recommendation Dashboard  
*  🚨    Risk & Impact Panel  

---

## 🛠️ Tech Stack

* **Python**
* **Streamlit**
* **Pandas & NumPy**
* **Matplotlib / Seaborn / Plotly**
* **Machine Learning (for predictions & optimization)**
